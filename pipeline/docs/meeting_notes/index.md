## Adding a new meeting

1. Duplicate the [template](./_template.md) into
   `meeting_notes/YYYY-MM-DD.md`.
2. Add a row to the archive table above.
3. Move any completed actions from the "Open actions" list to
   "Recently completed". Any new actions land in "Open actions" **and**
   on the meeting's own page.
